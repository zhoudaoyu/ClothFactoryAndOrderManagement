# ClothFactoryAndOrderManagement
This is the project for cloth product and order management with factory model.

I'd like to create on factory. It will product some kinds of cloth like shirts, shoes, trousers. And I think Sales is also important. 
So I want to build online sales for client as well. This requires order management and client center. 

I think we can separate that into serveral steps:
1. build each product and factory.
2. build order management.
3. build client relationship management.
